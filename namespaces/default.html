smart_resize_image()

smart_resize_image(mixed $file, mixed $width, mixed $height, mixed $proportional, mixed $output, mixed $delete_original, mixed $use_linux_commands, mixed $quality) : boolean|resource

easy image resize function

Parameters

mixed $file
  • file name to resize
mixed $width
  • new image width
mixed $height
  • new image height
mixed $proportional
  • keep image proportional, default is no
mixed $output
  • name of the new file (include path if needed)
mixed $delete_original
  • if true the original image will be deleted
mixed $use_linux_commands
  • if set to true will use "rm" to delete the image, if false will use PHP unlink
mixed $quality
  • enter 1-100 (100 is best quality) default is 100

Returns

boolean|resource
